# BRIEF: BRIEF-20260311-006-claude-github-smoke

## Context
Нужна короткая smoke-проверка GitHub-контура между Codex Mac и Claude Mac.

## Objective
Проверить, что Claude:
- читает `active_plan.md`;
- видит выданную задачу через репозиторий;

- умеет дать минимальное подтверждение приема задачи.

Claude должен ответить только фразой `Задачу принял` и не выполнять никаких иных действий.

## Constraints
- Без изменений runtime-кода.
- Без изменений orchestration state-файлов со стороны Claude.
- Без отчетов о выполнении.
- Без коммитов, PR и дополнительных файлов.

## Success Criteria
- Подтверждение `Задачу принял` получено.
- Репозиторий не изменен Claude.
- Active-plan дисциплина не нарушена.
