## Что сделано

Исправлен `infer_scene_profile()` в `python_scripts/training_conveyor.py` (lines 66–74).

**Причина бага:** substring-matching — `"bird"` содержит `"ir"` как подстроку, поэтому
`drone-bird-yolo` ошибочно классифицировался как `scene_profile="ir"`.

**Фикс:** добавлена helper-функция `_scene_tokens()`, разбивающая имя по разделителям
`[-_\s]+` и возвращающая множество токенов. `infer_scene_profile()` теперь сравнивает
целые токены через set-intersection вместо `in`-substring.

```python
def _scene_tokens(name: str) -> set:
    return set(re.split(r"[-_\s]+", name.lower()))

def infer_scene_profile(name: str) -> str:
    tokens = _scene_tokens(name)
    if tokens & {"ir", "thermal", "rgbt"}:
        return "ir"
    ...
```
