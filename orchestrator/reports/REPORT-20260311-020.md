## Что сделано

Добавлена brightness-based auto scene detection в `TrackerPipeline`.

### Алгоритм (TASK-020)

**config.py** — новые поля:
| Поле | Default | Описание |
|------|---------|----------|
| `AUTO_SCENE_DETECT` | False | Включить auto scene detection |
| `AUTO_SCENE_NIGHT_BRIGHTNESS_MAX` | 50 | mean Y < N → ночная сцена |
| `AUTO_SCENE_CONFIRM_FRAMES` | 30 | кадров подряд до переключения (hysteresis) |
| `AUTO_SCENE_SAMPLE_INTERVAL` | 10 | анализ каждые N кадров |
| `AUTO_SCENE_NIGHT_CONF` | 0.12 | CONF_THRESH при ночной сцене |
| `AUTO_SCENE_NIGHT_MOT_THRESH` | 12 | NIGHT_MOT_THRESH при ночи |
| `AUTO_SCENE_NIGHT_DIFF_THRESH` | 8 | NIGHT_DIFF_THRESH при ночи |

**pipeline.py** — метод `_adapt_auto_scene(frame)`:
- Запускается каждые `SAMPLE_INTERVAL` кадров
- Анализирует центральный кроп кадра (25-75% по обеим осям, игнорирует виньетирование)
- При `mean_brightness < NIGHT_BRIGHTNESS_MAX` → candidate='night'
- Требует `CONFIRM_FRAMES / SAMPLE_INTERVAL` подтверждений подряд
- На switch: динамически изменяет `cfg.CONF_THRESH`, `cfg.NIGHT_MOT_THRESH`, `cfg.NIGHT_DIFF_THRESH`
- На day restore: восстанавливает оригинальные значения (сохранены при init)

**app/main_gui.py** — `_apply_canonical_operator_mode('auto')`:
- Устанавливает `self._auto_scene_detect_enabled = True`
- В `_build_config()`: `cfg.AUTO_SCENE_DETECT = self._auto_scene_detect_enabled`
- Для 'day'/'night'/'ir' → `_auto_scene_detect_enabled = False`

### Поведение по умолчанию

`AUTO_SCENE_DETECT = False` — функция неактивна, все ручные режимы не затронуты.
Активируется только при нажатии кнопки «Авто».

## Риски

- Динамическое изменение `cfg.CONF_THRESH` mid-run меняет поведение YOLO уже следующего кадра. Это безопасно, но нужна ручная проверка стабильности.
- Threshold `mean_Y < 50` предварительный — может потребовать настройки на реальных ночных клипах.
- На IR/thermal сценах яркость может быть высокой → алгоритм не переключится в night mode. Для IR нужен отдельный heuristic (задача вне scope).

## Что осталось

- Ручная калибровка `AUTO_SCENE_NIGHT_BRIGHTNESS_MAX` на реальных ночных клипах
- Рассмотреть IR heuristic (отдельная задача)
