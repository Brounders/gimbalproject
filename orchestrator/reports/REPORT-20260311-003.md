# REPORT: TASK-20260311-003-package-import-decoupling

## Summary
Исправлена граница импорта пакета `src/uav_tracker/__init__.py`. Убран eager import `pipeline`, который транзитивно тянул `cv2`, `ultralytics` и `torch` при любом `import uav_tracker`. Заменён на lazy pattern через `__getattr__` с кэшированием в `globals()`. Обратная совместимость публичного API сохранена. Добавлен regression-тест `tests/test_package_import.py` на 3 проверки.
