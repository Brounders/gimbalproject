# Legacy Scripts

This directory stores archived prototype scripts.

- These files are preserved for historical reference and reproducibility.
- They are **not** production entrypoints for the current tracker.
- Production runtime uses:
  - `python main_tracker.py` (CLI)
  - `python tracker_gui.py` (Desktop GUI)
  - scripts under `python_scripts/`

Archived prototypes live in `legacy/prototypes/`.
Root-level wrappers remain only for backward-compatible launches.
