# PySide6 UI Playbook

## Когда использовать
- Любые изменения desktop UI.
- Упрощение операторского потока.
- Панели, header, left rail, HUD, video stage.

## Базовые правила
- UI не содержит бизнес-логику.
- Runtime/track logic не зависит от слоя отображения.
- Изменения staged, без переписывания окна с нуля.
- Сначала структура и стабильность, потом визуальная polish.

## Предпочтения проекта
- Минималистичный операторский интерфейс.
- Только нужная информация на видео-слое.
- Экспертные настройки отделены от базового операторского контура.
