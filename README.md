# Система сопровождения БПЛА

## Текущий операторский baseline
- `display_box_alpha = 0.40` принят как базовое значение для стандартных профилей.
- Для высокоманевренных целей добавлен профиль `fast` с `display_box_alpha = 0.30`.
- Плавность рамки теперь адаптивная: ядро автоматически подстраивает сглаживание по скорости и изменению размера цели.

## Профили в интерфейсе оператора
В левом блоке доступны два простых выбора:
1. `Режим камеры`: `Auto / Day / Night`
2. `Профиль сопровождения`: `Стандарт (0.40)` или `Fast (0.30)`

Рекомендация:
- `Стандарт` — базовая стабильность и читаемость;
- `Fast` — быстрее визуальная реакция на резкие маневры.

## IR‑фильтр шума (mini-gate)
В IR-режиме введен усиленный фильтр:
- при росте шумового уровня повышается порог подтверждения lock (`confirmation_frames_ir_noise`),
- для quality-gate применяется отдельный порог по неподтвержденной активности (`ir_noise_unverified_rate`).

Параметры задаются в YAML-профиле:
- `ir_mode_enabled`
- `ir_noise_threshold`
- `confirmation_frames_ir_noise`
- `ir_noise_unverified_rate`

## Ключевые пресеты
- `configs/operator_standard.yaml` — операторский baseline (0.40)
- `configs/fast.yaml` — fast-профиль (0.30)
- `configs/night_ir_lock_v2.yaml` — ночной/IR рабочий профиль
